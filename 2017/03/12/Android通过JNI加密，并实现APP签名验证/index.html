Android通过JNI加密，并实现APP签名验证

发表于 2017-03-12 | 分类于 学习 | | 阅读次数

一.前言：

移动端数据加密，需要将RSA加密公钥存储，考虑到安全，现将公钥存在.so文件中，为了防止有人将.so文件拷贝使用，因此在里面验证app的签名，对比签名信息，如果正确，返回正确公钥。
